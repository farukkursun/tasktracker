# React Tasktracker

Responsive Movie App made using React and Bootstrap.

<hr />

## What I Used



- React
- React-Bootstrap



### Installation Options

```
$ git clone https://github.com/farukkursun/movie-app
```

```
$ npm install
```

```
$ npm start
```

### Or

```
$ git clone https://github.com/farukkursun/movie-app
```

```
$ yarn
```

```
$ yarn start
```
